# linux-cybersec-journey
Studied root-level Linux commands as part of the Linux for Hackers YouTube series from "Network Chunk".

Covered in this session:
- sudo and su — privilege escalation basics
- chmod / chown — file permission and ownership management
- useradd / userdel — user account management
- passwd — password management
- systemctl — service management (start, stop, enable)
- kill / pkill — process termination
- apt / dnf — package management with root access
